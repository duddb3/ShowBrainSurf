# ShowBrainSurf
Function to quickly render values (stats, cortical thickness, etc.) onto brain surfaces, or to show atlas defined ROIs on the brain surface

Example: Cohen's D effect sizes mapped onto the Gordon Atlas functional parcellation (333 ROIs; Gordon EM, et al. Cerebral Cortex, Volume 26, Issue 1, January 2016, Pages 288â€“303, https://doi.org/10.1093/cercor/bhu239)

usage: > showbrainsurf(X);

where X is a 333x1 vector containing Cohen's D effect sizes.
